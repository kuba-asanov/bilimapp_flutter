# sanack_flutter

Flutter project - Sanack.

## Getting Started
Clone the project

#### Get dependencies
```
flutter pub get
```
#### Run project
```
flutter run 
```

### Used libraries:

- flutter_bloc
- freezed
- image_picker
- image_cropper
- get_it
- injectable
- auto_route
- flutter_screenutil
- dartz

### To build run this command:
```
sh scripts/build.sh
```

### To generate localization run this command:
```
sh scripts/lang.sh
```

To navigate to new page use:
```
Navigation.router.push(MyRoute());
```
