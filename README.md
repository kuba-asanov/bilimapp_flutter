# kodjaz_mobile

Flutter project - kodjaz.

## Getting Started
Clone the project

#### Get dependencies
```
flutter pub get
```
#### Run project
```
flutter run 
```

### Used libraries:

- flutter_bloc
- freezed
- get_it
- injectable
- auto_route
- flutter_screenutil
- retrofit
- dio
- hive
- code_text_field
- flutter_markdown
- flutter_smart_dialog

### To build run this command:
```
sh scripts/build.sh
```

### To generate localization run this command:
```
sh scripts/lang.sh
```

To navigate to new page use:
```
Navigation.router.push(MyRoute());
```
